## EXAMPLES :floppy_disk:

CHAR:
```C
_printf("Character:[%c]\n", 'H');
```
OUTPUT:
```C
CHARACTER:[H]
```

STRING:
```C
_printf("String:[%s]\n", "I am a string !");
```
OUTPUT
```C
String:[I am a string !]
```

INTEGER - unsigned int - double
```C
_printf("Numbers: [%i] [%d] [%u] ", 1, 2, 3);
```
OUTPUT
```C
Numbers: [1] [2] [3]
```
